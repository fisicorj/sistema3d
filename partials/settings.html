🖨️ Custo da Impressora por Hora

O sistema calcula automaticamente: depreciação/h + manutenção/h + energia/h. O valor da impressora, vida útil e potência ficam no cadastro de impressoras.

Usado apenas quando nenhum item ativo estiver cadastrado em 🔧 Manutenção. Com itens ativos, a soma deles substitui este valor automaticamente.
Selecione uma impressora na calculadora para ver o custo/hora.

💰 Regras de Preço

Percentual aplicado sobre filamento para suporte, purge e sobras.
Padrão usado na calculadora quando não especificado por pedido.
Ex.: 5% faz o sistema dividir o custo por 0,95 para cobrir impressões perdidas.
Aplicado sobre o lucro bruto para estimar o lucro líquido. Use 0 para ignorar.
Use 0 quando quiser cobrar embalagem apenas se uma embalagem for selecionada.
Aparece como barra de progresso no dashboard. Use 0 para ocultar.
Pedidos em aberto há mais que este prazo aparecem como alerta no dashboard.
Itens zerados por padrão: taxa de personalização e atendimento ao cliente. Use apenas quando houver criação/modelagem real.

Conecta ao broker MQTT local da impressora para exibir progresso, camadas e temperaturas diretamente nos cards de pedido. A impressora e o computador devem estar na mesma rede.

Encontre em: Menu da impressora → Settings → Device → Serial Number
Encontre em: Menu da impressora → Settings → WLAN → Access Code

🚚 Melhor Envio

Configure sua conta para cotar PAC, Sedex e transportadoras pela API. O token fica salvo apenas neste computador, em app_data/melhor_envio_config.json.

Por segurança, o token salvo não aparece na tela. Deixe vazio para manter o token atual.
Use vazio para deixar o Melhor Envio retornar todos os serviços disponíveis.
