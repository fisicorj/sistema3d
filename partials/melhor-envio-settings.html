🚚 Melhor Envio

Configure sua integração para cotar frete pelo CEP usando o servidor local.

🔑 Dados da integração

O token fica salvo somente neste computador. Para funcionar, abra o sistema pelo server.py, iniciar.sh ou iniciar_windows.bat, não direto pelo arquivo HTML.

Depois de salvo, o token não aparece novamente. Para trocar, cole outro token e salve.

📦 Como usar na calculadora

Na calculadora, informe o CEP de destino, peso/dimensões da caixa e clique em Cotar Melhor Envio. Se a API falhar, o sistema volta para a tabela local de frete por UF.
